# Acme Cloud Integration Service

This repository contains the core integration service used to connect internal
Acme tooling with third-party cloud providers.

## Overview

The service handles:
- Authentication with external APIs
- Telemetry ingestion
- Secure deployment automation

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/acme-org/acme-cloud-integration-service.git
cd acme-cloud-integration-service
npm install
